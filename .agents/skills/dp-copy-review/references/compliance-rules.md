# Compliance Rules — Copy Review

## Interdictions absolues

| Catégorie | Interdit | Exemple | Alternative |
|-----------|----------|---------|-------------|
| Income guarantee | Promettre un revenu spécifique | "Gagne 10K€/mois" | "Construis un système pour attirer des clients régulièrement" |
| Unrealistic promise | Promettre des résultats irréalistes | "100 clients en 1 semaine" | "Un système testé pour trouver tes premiers clients" |
| Fake urgency | Fausse urgence sans fondement | "Plus que 3 places !" (si faux) | Urgence réelle basée sur des faits |
| Fake scarcity | Rareté artificielle qui se reset | "Offre limitée" permanent | Offre avec vraie date de fin |
| Atypical testimonials | Résultats exceptionnels sans disclaimer | "J'ai fait 50K en 30 jours" | Ajouter "Résultats atypiques. Les résultats varient." |
| Unverifiable claims | Statistiques inventées | "97% de nos clients réussissent" | Claims basés sur des données réelles |
| Medical/Legal claims | Promesses de santé ou légales | "Guéris ton anxiété" | "Techniques de gestion du stress" |

## Obligations (si applicable)

| Obligation | Quand | Formulation |
|-----------|-------|-------------|
| Disclaimer résultats | Toute mention de résultats | "Les résultats varient selon l'effort et la situation." |
| Transparence prix | Produit payant mentionné | Prix clairement affiché |
| Nature du produit | Mention du produit | "Produit digital" / "Ebook" / "Guide en ligne" |
| Description honnête | Page de vente / landing | Ce qui est inclus, clairement listé |
| Identification pub | Publicité payante | Mentionner qu'il s'agit d'une pub (selon plateforme) |

## Mots bannis par plateforme

### Meta Ads (Facebook / Instagram)

| Catégorie | Mots/Expressions bannis |
|-----------|------------------------|
| Attributs personnels | "Tu es en surpoids", "Tu souffres de..." (ciblage perso) |
| Garanties financières | "Revenu garanti", "Deviens riche" |
| Before/After | Images avant/après (certaines niches) |
| Sensationnel | "Choquant", "Incroyable secret" |
| Crypto/MLM | "Opportunité de revenus passifs" |

### Google Ads

| Catégorie | Mots/Expressions bannis |
|-----------|------------------------|
| Superlatifs non prouvés | "Le meilleur", "N°1" (sans preuve tiers) |
| Caractères spéciaux excessifs | "!!!!", "***", majuscules excessives |
| Click-bait | "Cliquez ici !!!" |
| Contenu trompeur | Fausse similarité avec un résultat Google |

## Checklist de conformité par type

### Ad copy
- [ ] Pas d'attributs personnels (Meta policy)
- [ ] Pas de garantie de revenus
- [ ] Prix visible si produit payant
- [ ] Pas de superlatifs non prouvés (Google)
- [ ] Pas de fausse urgence

### Email copy
- [ ] Lien de désinscription mentionné / présent
- [ ] Expéditeur identifiable
- [ ] Objet non trompeur
- [ ] Pas de promesses de revenus

### Landing page copy
- [ ] Conditions générales accessibles
- [ ] Politique de remboursement claire
- [ ] Prix TTC affiché
- [ ] Disclaimer si témoignages

### Social post
- [ ] Mention #ad ou #sponsored si partenariat payé
- [ ] Pas de claims de santé non vérifiés
- [ ] Pas de before/after trompeur
