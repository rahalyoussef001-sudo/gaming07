# Voice & Tone Guide — DP Créateur

## Axes de Voix (échelle 1-10)

| Axe | Position | Description |
|-----|----------|-------------|
| Formalité | 3/10 | Informel mais pro. Comme un mentor, pas un prof. |
| Expertise | 8/10 | Sait de quoi il parle. Pas pédant. |
| Énergie | 6/10 | Dynamique sans être surexcité. Pas de "!!!". |
| Empathie | 7/10 | Comprend la galère du lecteur. Pas condescendant. |
| Directivité | 9/10 | Dit exactement quoi faire. Pas d'ambiguïté. |
| Honnêteté | 9/10 | Avoue les limites. Pas de promesses vides. |

## Patterns d'écriture

### Le hook de section
```
❌ "Dans cette section, nous allons explorer les différentes façons de..."
❌ "Il est important de comprendre que..."
✅ "Tu perds des clients parce que ton offre ressemble à toutes les autres. Voici comment te différencier en 30 minutes."
✅ "La prospection, c'est un jeu de nombres. Voici les nombres qui comptent."
```

### L'instruction d'action
```
❌ "Il serait bénéfique de considérer l'envoi de messages directs."
❌ "Pensez à contacter régulièrement vos prospects."
✅ "Envoie 20 DMs par jour. Pas 5. Pas 10. Vingt."
✅ "Ouvre Notion. Crée un tableau avec 3 colonnes : Prospect, Date, Statut."
```

### Le caveat honnête
```
❌ "Les résultats sont garantis si vous suivez le programme."
❌ (rien — pas de mise en garde)
✅ "Est-ce que ça marche à tous les coups ? Non. Ça dépend de ta niche, de ton prix, et de ta capacité à exécuter. Mais c'est la méthode qui donne les meilleurs résultats avec le moins d'efforts."
✅ "Attention : si ton offre n'est pas claire, aucune quantité de DMs ne va la sauver. Retourne à la Section 2."
```

### La transition entre sections
```
❌ "Maintenant que nous avons vu X, passons à Y."
❌ "Le prochain chapitre traitera de..."
✅ "Ton offre est claire. Ton message est prêt. Il est temps de le mettre devant les bonnes personnes."
✅ "Tu sais quoi dire. Maintenant, à qui le dire."
```

## Mots interdits

| Catégorie | Interdits | Alternatives |
|-----------|-----------|--------------|
| Fluff motivationnel | "Tu peux le faire !", "Crois en toi", "N'abandonne jamais" | (supprimer — pas d'alternative) |
| Vague | "Sois authentique", "Apporte de la valeur", "Sois proactif" | Donner l'action spécifique |
| Académique | "Il convient de", "Il est à noter que", "Selon les études" | Énoncer le fait directement |
| Jargon creux | "Levier", "Synergies", "Écosystème", "Paradigme" | Utiliser le mot simple |
| Superlatifs vides | "Incroyable", "Extraordinaire", "Révolutionnaire" | Donner le chiffre ou le résultat |
| Passif | "Des DMs doivent être envoyés", "Il est recommandé de" | "Envoie des DMs", "Fais X" |

## Mots encouragés

| Catégorie | Exemples |
|-----------|----------|
| Action directe | "Fais", "Envoie", "Crée", "Ouvre", "Écris", "Lance" |
| Spécificité | "20 DMs/jour", "en 14 jours", "3 étapes", "script de 4 lignes" |
| Honnêteté | "Ça dépend de", "Résultats variables", "Pas de miracle" |
| Concret | "Template prêt à copier", "Checklist en 5 points", "Script exact" |
| Transition action | "Maintenant", "Prochaine étape", "Passe à" |

## Règles de formatage

1. **Paragraphes** : 4-5 lignes max. Si plus long, couper.
2. **Listes** : Préférer les listes à puces pour les étapes et actions.
3. **Gras** : Pour les mots-clés et les points d'attention. Pas pour des phrases entières.
4. **Italique** : Rarement. Pour les citations ou les nuances.
5. **Majuscules** : Jamais de phrases en full caps. JAMAIS.
6. **Emojis** : Pas dans le corps du texte. Acceptables dans les titres h4 de blocks.
7. **Questions rhétoriques** : Max 1 par section. Seulement pour nommer un ressenti du lecteur.
